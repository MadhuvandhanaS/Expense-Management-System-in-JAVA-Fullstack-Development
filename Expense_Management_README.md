# Expense Management System - Java Full Stack Project

## 📌 Project Overview
The Expense Management System is a full-stack Java application developed to help users manage daily expenses efficiently.

This project allows users to:
- Add expenses
- Update expense details
- Delete expense records
- Track spending history
- Manage financial data using database integration

This project demonstrates Java Full Stack Development concepts including frontend development, backend logic, database connectivity, and CRUD operations.

---

## 🚀 Features
- Add New Expenses
- Update Existing Expenses
- Delete Expenses
- View Expense Records
- Expense Categorization
- Database Connectivity
- Simple & User-Friendly Interface
- CRUD Operations

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/MadhuvandhanaS/Expense-Management-System-in-JAVA-Fullstack-Development.git
```

### 2️⃣ Open in IDE
- Import the project into Eclipse or IntelliJ IDEA

### 3️⃣ Configure Server
- Configure Apache Tomcat Server

### 4️⃣ Database Setup
- Create MySQL database
- Import SQL file
- Update database username & password in project files

### 5️⃣ Run the Project
- Start Tomcat Server
- Run the application
